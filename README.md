# Paciente-Student-Portfolio-for-COSC111

This portfolio showcases my learning journey and projects in **Internet of Things (IoT)**, where I explored the integration of **sensors, actuators, and microcontrollers** to create interactive systems.  

The learning process began with **simple Arduino projects**, such as basic LED control and running light sequences, and gradually advanced to **more complex applications**, including **sensor-based fire alarms, Python-Arduino integration, bi-directional communication, and web-controlled IoT systems using FastAPI**. Each project introduced new challenges and concepts, helping me strengthen both my **hardware and software skills**, as well as my understanding of **IoT system design**.  

All projects were completed in **collaborative group settings**, working with teams of **3, 5, and 6 members**, each presenting unique dynamics, problem-solving approaches, and learning opportunities. In total, this portfolio documents **9 completed projects**, demonstrating my growth from foundational IoT experiments to full-stack, interactive systems.
